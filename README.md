# 🐍 Python Fundamentals – Regular Expressions

This section contains tasks and exercises related to **Regular Expressions (Regex)** from the **Python Fundamentals** course.

Regular expressions are powerful tools used for **pattern matching and text processing**. They are commonly used for extracting information from strings, validating input, and manipulating text data.

## Topics Covered

In this folder you will find several groups of tasks designed to practice different aspects of regex in Python:

### Lab
The **Lab** tasks focus on the basics of regular expressions.  
They introduce fundamental regex syntax and simple pattern matching using Python's `re` module.

Topics include:
- Basic pattern matching
- Using `re.findall()`, `re.search()`
- Character classes
- Groups and capturing

### Exercise
The **Exercise** section contains more complex problems that require combining multiple regex techniques with Python logic.

These tasks typically involve:
- Extracting structured data from text
- Using capturing groups
- Processing matches and building program logic
- Real-world style parsing problems

### Bonus Exercise
The **Bonus Exercise** tasks are optional but more challenging problems that help deepen the understanding of regex and problem solving.

They usually require:
- More advanced regex patterns
- Edge case handling
- Combining regex with data structures and algorithms

### ChatGPT Tasks
This section contains **additional practice problems and experiments created with the help of ChatGPT**.

These tasks are used to:
- Explore different regex approaches
- Test understanding of regex behavior
- Practice alternative solutions and optimizations
